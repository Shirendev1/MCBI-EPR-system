# MCBI – Монголын EPR систем

Энэ repository нь Монголд хэрэгжүүлэх тойрог эдийн засаг, батарейн хаягдал болон EPR системийн эхний баримт бичиг, загвар, өгөгдлийн бүтцийг хадгална.

## Зорилго

Монголд батарей болон бусад хаягдлын хариуцлагатай цуглуулалт, дахин ашиглалт, дахин боловсруулалтын систем бий болгох.

## Агуулга

- EPR бодлогын санал
- Тойрог эдийн засгийн системийн загвар
- Өгөгдөл цуглуулах бүтэц
- Тайлагналын загвар
- Диаграм ба хавсралтууд

## Гол санаа

EPR бол зөвхөн хог хаягдлын тухай биш.  
Энэ бол үйлдвэрлэгчийн хариуцлага, өгөгдөл, санхүүжилт, дэд бүтэц, хэрэглэгчийн зан төлөвийг нэг системд холбох арга юм.
